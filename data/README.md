# Dataset Information

The datasets used in this project Open University Learning Analytics Dataset (OULAD).

Due to their large size, they are not hosted directly in this repository's main view but are required for the analysis.

## Source
- **Name**: Open University Learning Analytics Dataset (OULAD)
- **Source**: [OULAD Dataset](https://analyse.kmi.open.ac.uk/open-dataset)


## File Structure
- `assessments.csv`
- `courses.csv`
- `studentAssessment.csv`
- `studentInfo.csv`
- `studentRegistration.csv`
- `studentVle.csv`
- `vle.csv`

Please download these files and place them in this `data/` directory to run the notebook locally.
